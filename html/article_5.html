Chap 1 Models of Computation

1.1 Motivation

An algorithm tells a computer what do.

An algorithm can be written in 3 ways:

  1. free text
  2. pseudocode and
  3. word RAM instructions

Free text is used to present a question, pseudocode is used to make it easier to implement an algorithm and word RAM is used to present the algorithm in the form of a well-defined a mathematical object to be used for mathematical analysis.

1.2 Word RAM

Word RAM is also used to show how physical computers use data in an easy to understand way

It uses words of fixed size instead of bits. A word is is at least as big a memory address or integer.

Word RAM is made up of: the memory and CPU. Memory is a large array of words. With each word being accessible via its address . The CPU is where the instructions are executed.

The primitive operations are:

  • Assign Constants
  • Copy
  • Copy via Indirect Access
  • Arithmetic on words
  • Bitwise operations
  • Shifts
  • Comparisons
  • Control flow

Basic operations are O(1).

Every operation has a precise and agreed upon meaning and cost.

There is no questions about what is a step, the type of memory access allowed, or the size of the numbers.

Benefits

Word RAM programs are independent of specific processor architecture.

Word RAM model can be used as a reference point for analysis because any algorithm can be written as a word RAM program.

Drawbacks

It is hard to understand because there are more instruction as the program gets bigger and it adds more details that are not related to what the program is mainly about.
