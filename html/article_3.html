Explain definitions

There are definitions that are needed to be defined before I tell you the definition of XAI and get into the rest of the article. In this section.

1. Explainability

Means the ability to describe the model’s black-box behaviour. There are 4 maxims that must be followed by explanations: 

1. Quality:  Finds out the correct explanation.
                        2. Quantity: Deliver the appropriate amount of information that is informative.
                        3. Relation: An explanation must contain only information relevant to the discussion. 
                        4. Manner: the information should be  delivered in this manner: 
                            1. no ambiguous words. 
                            2. no ambiguous explanations. 
                            3. the explanation should be concise. 
                            4. information should be in order. 
                        5. Context-oriented: Explanations should depend on type of user, i.e. developer, regulator, customer, etc.

2. Interpretability

The ability to explain or to provide the meaning of something in understandable terms to a human.

3. Transparency

a model is considered to be transparent if by itself it is understandable.

4. Understandability/ intelligibility

Explains the characteristic of a model to make a human understand how the model works – without explaining its internal structure or the algorithms.

5. Comprehensibility

The ability to represent what a model has learned in a human understandable fashion

6. Fairness

A model’s ability to make unbiased decisions.

7. Robustness

The sensitivity of the system’s output to a change in the input.

8. Causality

Measures the change in the predicted output.

9. Satisfaction

The ability to improve the usability and utility of the model.

10. Stability

The ability to provide comparable explanations for inputs that are similar.

Black-box, white-box and grey-box models

A white-box or a glass-box model are models that can be understood by glancing at the summary or parameters of the model without the need for an external model to provide an explanation.

A grey-box model are models that users can interpret at some degree if they are carefully designed.

A black box model is one where you just don't get how it works.

Defining XAI

There is no standard definition of explainable AI. For this article I will adopt the definition of DARPA, XAI aims to ‘‘produce more explainable models, while maintaining a high level of learning performance (prediction accuracy); and enable human users to understand, appropriately, trust, and effectively manage the emerging generation of artificially intelligent partners’’.

Explainability methods

There are a variety of methods with their advantages and disadvantages.

Surrogation

A separate model is created to explain the black-box decision either locally or globally and this interpretable.

Locally Interpretable Model-Agnostic Explainer (LIME)

LIME finds out what happens to predictions when different inputs are fed into the model. LIME creates a new dataset using changed samples and the black-box model’s predictions. LIME then uses this dataset to build an interpretable model that is weighted by the sampled instances’ closeness to the instance of interest.

Local Rule-based Explanation (LORE)

Uses an ad-hoc ‘‘genetic algorithm’’ to generate a set of neighbour instances of the given instance, from which a decision tree classifier can be made.

Cluster Representatives with LIME (CluReFI)

First, LIME is used to describes the representation of a cluster once the data has been clustered. After allocating an unknown data instance to the closest cluster, cluster assignments are visualized. Then, CluReFI visualizes each cluster’s feature validity ranges for the most essential features. CluReFI shows the user the most significant aspects that contribute to the class for their representation

Submodular Pick (SP)-LIME.

SP-LIME is a strategy for identifying the must important model predictions. The number of events to examine is chosen to be as large as possible so that the model can be understood.
