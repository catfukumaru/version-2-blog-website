Chap 1 Models of Computation

1.3 Turing Machines

  • The main value of Turing machines, compared to the word RAM, is their mathematical simplicity.

  • The Turing machines is compared to a worker that has a paper divided into squares. He's job is to to puts a symbol in a square. He can only look at one square at a time and it has no memory of the previous square. The worker has a set of instructions that define which symbol goes in the square it is on.

  • Qualities of Turing machines:

    • It's mathematically simplistic because it has a few operations and it has no advanced mathematical assumptions.
    • Algorithms that can run in the word RAM can run in the Turing machine.
    • To design algorithms on a Turing machine it is best to use the multi-tape version.

1.4 Real Models of Computation

Real models of computation are abstract models were real numbers(1,2,3) are treated as basic objects of computation, instead of being floating point numbers.

Pros

  • No worries about rounding, precision loss or point errors.
  • Simplifies problems.

Cons

  • They do not reflect the reality of working in physical machines.

A word RAM can be extended to real by adding more storage to store real numbers and arithmetic operations are added ot the instruction set.

The same can be said for a real turning machine, by adding more tape for real numbers and the instruction set is given rules for real numbers.

1.5 Cheating with the Real RAM

Cheating means encoding a large amount of data into one real number and manipulating it using constant-time operations, which reduces computations that would take exponential time into polynomial time.

One option of cheating would be rounding.
